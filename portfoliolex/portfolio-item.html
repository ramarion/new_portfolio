Tamagotchi Game

An Interactive Game

I was tasked to create a tamagotchi game that a user would be able to play with and have fun. The pet would be able to convey emotions and be fed and also speak to the user.

The User has their name taken in and will be able to start the game.

The pet then returns a message to start the game and it has 60 seconds to live. During that the time the user can feed it to add calories to the counter to keep it alive. Each time the pet returns a compliment and convey an emotion back to the user.
